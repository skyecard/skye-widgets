# skye-widgets
A collection of skyecard widgets

# How to build
```
cd /to/this/directory
npm install
npm run build
```

# Dependencies
This widget builder uses the following key technologies:
* nodejs
* npm
* webpack
* typescript
* jquery
* jquery-modal
